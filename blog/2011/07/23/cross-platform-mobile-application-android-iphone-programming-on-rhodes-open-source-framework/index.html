Cross Platform Mobile Application Android iPhone programming on Rhodes Open Source Framework

Posted by Star on July 23, 2011 | 0 comments

I got this message, ERROR!!! Wrong Android API version, while
I’m trying to build a simple android app with Rhodes, the major open source cross platform framework for mobile development.

rhodes app helloworld
cd helloworld
rake run:android

I’m using the rhodes-3.0.2 gem and Android 3.1 SDK (r12).

When I removed the 3.1 sdk and using 3.0 sdk, it’s working!!

but meeting an other issues:


star@Ubuntu:~/helloworld$ rake run:android

the problem is current Rhodes 3.0.2 gem supports r5b NDK, but I’m using r6b NDK:

It’s possible to download it by these links:

Windows: http://dl.google.com/android/ndk/android-ndk-r5b-windows.zip
MacOS: http://dl.google.com/android/ndk/android-ndk-r5b-darwin-x86.tar.bz2
Linux: http://dl.google.com/android/ndk/android-ndk-r5b-linux-x86.tar.bz2
